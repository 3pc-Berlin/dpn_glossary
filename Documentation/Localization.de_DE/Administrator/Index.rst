Administrator Anleitung
=======================

Zielgruppe: **Administratoren**

.. _admin-installation:

Installation
------------

#. Installieren Sie die Extension über den Extension Manager

#. Fügen Sie das statische TypoScript zu Ihrem Seiten-Template hinzu

#. Erstellen Sie eine Listen- & detailpage für Begriffe

#. Konfigurieren Sie die Uids in den Konstanten damit die Detailseite für den Parser auflösbar ist

   - Setze die Ordner Seiten uid (persistence.storagePid)
