Administrator Anleitung
=======================

Zielgruppe: **Administratoren**

.. _admin-installation:

Installation
------------

1. Installieren Sie die Extension über den Extension Manager

2. Fügen Sie das statische TypoScript zu Ihrem Seiten-Template hinzu

3. Erstellen Sie eine Listen- & detailpage für Begriffe

4. Konfigurieren Sie die Uids in den Konstanten damit die Detailseite für den Parser auflösbar ist
