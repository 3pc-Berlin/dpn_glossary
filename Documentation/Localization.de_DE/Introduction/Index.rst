Einleitung
==========

Was tut die Extension?
----------------------

Die Extension lässt Sie ein simples Glossar für Ihre Website einrichten.
Zusätzlich bietet der automatische Parser die möglichkeit Begriffe auf Ihrer Website zu einer
Detailseite zu verlinken.

------------------------

+ Automatisches parsen von Begriffen auf Seiten
